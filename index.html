Web Portfolio

Final Project, Intro to Programming Class

This is a website that I created with three other Epicodus students as our final project for Intro to Programming. It is a website where a user can browse or search for parks in the Portland area and leave reviews for the parks. To create this site, we used HTML and CSS, JavaScript, jQuery, Mocha and Chai.

Ping Pong

This is my week three of Intro to Programming code review. It is a website where a user can input a number and have a range of numbers from 1 to their entered number returned in a list. Numbers divisible by 3 are replaced with the word "ping", numbers divisible by 5 are replaced with "pong", and numbers divisible by both are replaced with "pingpong".

Vacation Destination Quiz

A website that I made for my second code review at Epicodus. It is a quiz that allows a user to answer a few questions and suggests a vacation destination based on their answers.
